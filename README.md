# 今日待办

一个简单美观的待办事项应用，使用Next.js和Tailwind CSS构建。

## 功能

- 添加、完成和删除待办事项
- 使用本地存储保存数据
- 响应式设计，适配各种屏幕尺寸
- 美观的用户界面

## 技术栈

- Next.js
- React
- TypeScript
- Tailwind CSS
- localStorage

## 安装

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器中访问 [http://localhost:3000](http://localhost:3000)。

## 构建

```bash
# 构建生产版本
npm run build

# 运行生产版本
npm start
```
